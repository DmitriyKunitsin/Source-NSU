# Расчёт установившейся ошибки для пункта 4.6

## Исходные данные (по методическим указаниям)

Для пункта 4.6:

- Передаточные функции:

$$W_1(s) = \frac{K_1}{T_1^2 s^2 + 2d T_1 s + 1}$$

$$W_2(s) = \frac{K_2}{T_2 s + 1} \quad \text{(апериодическое звено, заменившее интегратор)}$$

- Сигналы:

$$r(t) = r_0 \cdot 1(t), \quad m(t) = m_0 \cdot 1(t)$$

В изображениях по Лапласу:

$$R(s) = \frac{r_0}{s}, \quad M(s) = \frac{m_0}{s}$$

## Общее выражение для ошибки $E(s)$ (из стр. 3 лабораторной работы)

$$E(s) = \frac{1}{1 + W_1(s)W_2(s)} R(s) + \frac{(-1)W_2(s)}{1 + W_1(s)W_2(s)} M(s)$$

Подставляем $R(s)$ и $M(s)$:

$$E(s) = \frac{1}{1 + W_1W_2} \cdot \frac{r_0}{s} - \frac{W_2}{1 + W_1W_2} \cdot \frac{m_0}{s}$$

## Установившаяся ошибка (теорема о конечном значении)

$$e_0 = \lim_{t \to \infty} e(t) = \lim_{s \to 0} s E(s)$$

$$e_0 = \lim_{s \to 0} \left[ \frac{r_0}{1 + W_1(s)W_2(s)} - \frac{m_0 W_2(s)}{1 + W_1(s)W_2(s)} \right]$$

## Вычисление пределов при $s \to 0$

$$W_1(0) = K_1, \quad W_2(0) = K_2$$

Следовательно:

$$e_0 = \frac{r_0}{1 + K_1 K_2} - \frac{m_0 K_2}{1 + K_1 K_2}$$

## Итоговая расчётная формула

$$\boxed{e_0 = \frac{r_0 - m_0 K_2}{1 + K_1 K_2}}$$

$$\boxed{e_0 = \frac{1 - 2 * 1}{1 + 2 * 2}}=\boxed{\frac{-1}{5}}=\boxed{-0.2}$$
